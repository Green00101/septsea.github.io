Whether the propositions are true or false

Thursday, January 23rd 2020, 8:00:00 am math

May everything be better! Beware of 2019-nCov!

There are five propositions:

  • A: Both C and D are false.
  • B: Both A and E are false.
  • C: Both B and D are false.
  • D: Both C and E are false.
  • E: Both A and B are false.

Determine whether the propositions are true or false.

Here is a solution.

If A is true:

C is false. D is false.
C is false -> B is true & D is true
           or B is true & D is false
           or B is false & D is true
Since D is false, B must be true.
D is false -> C is true & E is true
           or C is true & E is false
           or C is false & E is true
Since C is false, E must be true.
A contradiction!

So A must be false. Then

   C is true & D is true
or C is false & D is true
or C is true & D is false
That both C and D are true is a contradiction.
That both B and E are true is a contradiction.

If C is true -> D is false:
-> B is false -> E is true

If C is false -> D is true -> E is false -> B is true

Conclusion:

(A, B, C, D, E) = (F, F, T, F, T)

or

(A, B, C, D, E) = (F, T, F, T, F)

Certainty: A is false.
