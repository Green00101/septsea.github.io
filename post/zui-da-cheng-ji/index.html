最大乘积

Friday, January 17th 2020, 8:00:00 am math

我, septsea, 又来水了。Wow, I can really water experience.

有一些正整数和为 n, 求它们最大乘积。

我们简单分析 e 下。

首先, 如果有 r 个 1, 那么就相当于 r - k 个 > 1 个数在相乘 (r 是数字个数)。2 * (w - 1) 不比 1 * w 香?

其次, 对于一个数 w 而言, 2 * (w - 2) = w + (w - 4), 3 * (w - 3) = w + (2w - 9), 所以, 不小于 5 的数 w 应当拆开。

4 可以写成 2 + 2 (当然也可以写成 2 * 2), 所以 WLOG (您觉得这该怎么读呢?), 把 4 拆为 2 个 2。

对于 6 而言, 6 = 2 + 2 + 2 = 3 + 3, 因为 2 * 2 * 2 < 3 * 3, 所以每 3 个 2 都应用 2 个 3 代替。

所以, 只要 n >= 2, 为了使乘积最大, 绝对不应出现 1, 尽可能用 3 填满, 剩下的用 2 填。

n = 1 时没得说, 乘积就是 1。n = 2 时, 1 个 2 就可, 从而最大乘积为 2。n = 3 时, 1 个 3 就可, 从而最大乘积为 3。n = 4 时, 2 个 2 就可, 从而最大乘积为 4。

n >= 5 时, 尽可能用 3 填满。设 n = 3q + r, 0 <= r <= 2。

r = 0 意味着可以全部用 3 填满, 从而最大乘积为 3 * 3 ^ (q - 1)。

r = 1 时应当用 q - 1 个 3 与 2 个 2 填满, 从而最大乘积为 4 * 3 ^ (q - 1)。

r = 2 意味着可以用 q 个 3 与 1 个 2 填满, 从而最大乘积为 6 * 3 ^ (q - 1)。

q = [n / 3], r = n - 3q, [x] 是不超过 x 的最大整数。
